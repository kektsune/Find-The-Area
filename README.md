Find The Area is a tool built using Python in Visual Studio Code that allows users to find the area of a shape based on their choice and plug in missing values.

Features:

Pictures of each shape
Submit button and text boxes to plug in answers
Calculations

IMPORTANT:
This program is only known to run in windows thus far. If you run it in macOS or Linux there's a chance it may not work.

This is the first release of Find The Area. Bug reports and feedback are welcome via the Issues tab.
